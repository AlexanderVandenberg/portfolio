Project 4:
Data Entry System

March 2022

Project Goal: Create a user interface for entering red tag data using Excel and VBA


Challenges

  • Although I had experience with Excel, I would be learning VBA from scratch
  • The old system was based in C++ and crashed a lot. The new system had to be based in Excel for greater stability
  • For each red tag, the system needed to be able to record nine pieces of data, reset all entries, and save all entries
  • The graphical user interface (GUI) needed to show the data for the most recently entered red tags

Design Steps

  • Created an Excel sheet that would hold all the red tag data. I added check boxes to the columns that don't have open-ended responses and filters to every column
  • Created a seperate sheet for a button that launches the GUI when clicked
  • Designed multiple iterations of a GUI that meets the requirements mentioned in the challenges section
  • Used VBA to link the GUI to the red tag data sheet

Results

  • Moving from C++ to Excel resulted in the new system having less bugs and crashes
  • Since most employees know Excel better than C++, the system is easier to use and modify
  • This system holds data from 1318 red tags as of September 2022
